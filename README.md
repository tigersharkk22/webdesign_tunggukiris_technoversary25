Nama Project: SMANDA AdiwiyatVerse

Deskripsi Project
-Website SMANDA AdiwiyataVerse adalah sebuah platform digital yang dibuat untuk mendukung program sekolah berwawasan lingkungan. Situs ini berfungsi sebagai pusat informasi dan edukasi tentang berbagai kegiatan hijau yang dilakukan di sekolah, seperti artikel sains, aksi peduli lingkungan, kalkulator jejak karbon, dan program “Pohon Rupiahmu” yang mendorong kontribusi nyata untuk pelestarian alam. Melalui tampilan yang modern dan konten yang informatif, website ini mengajak siswa, guru, dan masyarakat untuk lebih sadar terhadap isu lingkungan serta berpartisipasi dalam upaya menjaga bumi. SMANDA AdiwiyataVerse dirancang sebagai ruang interaktif yang menggabungkan teknologi dengan semangat hijau, sehingga dapat menjadi sarana pembelajaran sekaligus kampanye digital untuk keberlanjutan lingkungan di masa depan.


Cara Instalasi & Menjalankan Project

1. Clone repository:
   ```
   git clone https://github.com/tigersharkk22/SMANDA-ADIWIYATAVERSE.git
   ```

2. Masuk ke folder project:
   ```
   cd SMANDA-ADIWIYATAVERSE
   ```

3. Jalankan website secara lokal:
   Buka file index.html dengan browser
   (double click atau klik kanan → Open With Browser)

4. Edit project (opsional):
   Buka folder ini di VS Code atau editor lain
   lalu edit file HTML/CSS/JS sesuai kebutuhan.

5. Deploy ke GitHub Pages (opsional):
   * Push perubahan ke GitHub
   * Buka Settings → Pages → pilih branch `main` → root (`/`)
   * Website akan otomatis aktif.
